The Greptile MCP Server provides AI assistants and applications with direct access to Greptile's comments and context through 13 tools across 4 categories.

<Note>
**New to MCP?** Start with [Authentication](/mcp/authentication) → [Custom Context](/mcp/custom-context) → [Comment Resolution](/mcp/comment-resolution)
</Note>


## Quick Start

<Steps>
    <Step title="Get API Key">
        Sign up at [app.greptile.com](https://app.greptile.com/signup) and get your API key.
    </Step>
    <Step title="Add to Claude Code">
        ```bash
        claude mcp add --transport http greptile https://api.greptile.com/mcp \
          --header "Authorization: Bearer YOUR_GREPTILE_API_KEY"
        ```
    </Step>
    <Step title="Test Connection">
        Ask Claude: "What custom context does my organization have?"
    </Step>
</Steps>
