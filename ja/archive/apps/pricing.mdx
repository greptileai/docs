**要点** グレプタイルのWebチャットは、無料または無制限利用の定額月額プランです。グレプタイルのAPIおよびPRレビューなどの連携はAPIコールごとの従量課金で、新規ユーザーには150ユニットが無料で付与されます。

## グレプタイルチャット

グレプタイルチャットは、グレプタイルとやり取りできるWebベースのチャットインターフェースです。利用可能なプランは3つあります:

| プロパティ | Free | Pro |
| -------- | ------------------- | ------------------- |
| 価格     | $0 ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ ‌ |      $20/席/月 |
| リポジトリサイズ上限      | 10MB       |   ∞     |
| プライベートリポジトリ     | -      |   ✓    |
| チームの追加    | -      |   ✓    |
| セッション共有     | -      |   ✓ |
| チャットで複数リポジトリ     | -      |   ✓    |
| メールサポート     | -      |   48時間    |
| 共有Slackチャンネル     | -      |   -    |
| SOC 2 準拠     | -      |   -    |


<Note>ProおよびBusinessの両プランはSOC 2 準拠ですが、レポートの共有、セキュリティレビューの実施、その他のセキュリティ詳細の提供はBusinessプランのお客様に限ります。</Note>

## グレプタイル API

グレプタイル API の料金は「ユニット」単位で、1ユニットあたり $0.15 です。料金が発生するエンドポイントは `POST /query` のみです。 

`POST /query` の呼び出しは 1 ユニットです。

バグの診断などの複雑なタスクに適した、より大規模で高性能なモデルを使用するため、パラメータ `genius=true` を付与した `POST /query` の呼び出しは 3 ユニットです。

## Greptile Code Review Bot

グレプタイルのコードレビューボットの料金は、変更されたファイル1件あたり$0.45で、開発者1人あたり月額$50を上限とし、上限超過分の利用は無料です。

### 例

チームに開発者が4人いて毎週多くのPRを作成し、さらにエンジニアリングマネージャーがその月に5ファイル変更のPRを1件開く場合、その月の請求額は次の金額を超えません：

$$50 * 4 + 0.45 * 5 = 202.25$$


ご不明な点は support@greptile.com までお問い合わせください。
