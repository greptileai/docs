デフォルトでは、Greptile は新規に開かれた **すべての PR** をレビューします。ユーザーが PR に `@greptileai` とコメントした場合もレビューを行います。特定のトリガーで PR をレビューするように設定することもできます。

### フィルター

Greptile を、特定のフィルターに一致する PR のみレビューするように設定できます。フィルターは [ダッシュボード](https://app.greptile.com/review) または [`greptile.json` ファイル](/ja/code-review-bot/greptile-json)で設定できます。

| Filter By | Description                           |
| --------- | ------------------------------------- |
| `labels`  | 特定のラベルを持つ PR を含める／除外する |
| `branches` | 特定のブランチへの／からの PR を含める／除外する |
| `author` | PR の作成者に基づいて含める／除外する |
| `keywords` | タイトル、説明、または直近のコミットメッセージに特定のキーワードを含む PR を含める／除外する |

### 手動でトリガーする

PR に `@greptileai` とコメントすることで、Greptile に手動でレビューをトリガーできます。
