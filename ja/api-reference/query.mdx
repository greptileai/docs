---
title: "リポジトリのクエリ"
openapi: "POST /query"
---

コードベースに関する自然言語のクエリを送信すると、関連するコード参照（ファイルパス、行番号など）のリストとともに、自然言語の回答が返ってきます。

1. 「このコードベースの認証はどのように機能していますか？」
2. 「コードベースのコンテキストを用いて、JIRA チケットの説明を生成してください」
3. 「リポジトリ内の既存の関連抽象化を使って、このコードスニペットを書き換えてください」

注意: Greptile がリポジトリをクエリできるようにするには、事前にインデックス作成が必要です。詳細は [Index Repository](/ja/api-reference/index) を参照してください。
